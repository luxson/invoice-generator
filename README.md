Create professional invoices quickly and export them as PDF.

Der **KaLux Invoice Generator** ist eine moderne Webapplikation zur schnellen Erstellung, Vorschau und Generierung von professionellen Rechnungen.
Ideal für Freelancer, Agenturen und Unternehmen.

---

## Demo

* Erstellung von Rechnungen in Sekunden
* Live Vorschau
* PDF Export
* Automatische Steuerberechnung (Schweizer MWST)
* Modernes UI

## Installation

Clone repository:

```bash
git clone https://github.com/YOUR_USERNAME/kalux-invoice-generator.git
cd kalux-invoice-generator
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open:

```
http://localhost:3000
```
